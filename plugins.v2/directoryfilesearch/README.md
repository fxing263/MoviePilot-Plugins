# 目录文件搜索删除

MoviePilot V2 插件，用于在一个配置目录内递归搜索普通文件，并对单条搜索结果执行安全删除。

## 功能

- 按文件名或相对路径进行不区分大小写的包含搜索。
- 搜索和删除均在后台线程池执行，管理页通过轮询展示任务状态。
- 搜索结果分页展示文件名、相对路径、大小和修改时间。
- 支持单条、多选、当前页全选及全部搜索结果全选。
- 每次删除必须先生成整批预览，再勾选确认并提交后台任务。
- 确认令牌绑定文件设备号、inode、大小和纳秒修改时间，状态变化后拒绝删除。
- 只删除配置根目录内的普通文件，不删除目录、符号链接或越界路径。
- 后台串行删除真实源文件并逐项重新枚举父目录，只有确认目标名称不存在才标记成功。

## 默认配置

| 配置 | 默认值 |
| --- | --- |
| 启用插件 | 否 |
| 搜索目录 | 空，必须由用户配置 |

搜索关键词不能为空，单次最多保留前 1000 条匹配结果。插件不执行批量删除，也不自动删除空目录。
